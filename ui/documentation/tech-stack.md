# Tech Stack

- **Framework:** Angular 20
- **Language:** TypeScript
- **Routing:** Angular Router
- **Forms:** Angular Reactive Forms
- **HTTP:** Angular `HttpClient`
- **State:** Angular signals for local auth state
- **Styling:** Global CSS variables plus component-scoped CSS
- **Build tooling:** Angular CLI / Angular build system
- **Package manager:** npm
- **Testing:** Karma and Jasmine, as generated by Angular CLI

No UI component library is installed yet. This keeps the first implementation easy to refactor once the product direction is clearer.
