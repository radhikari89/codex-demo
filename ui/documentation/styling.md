# Styling

The current styling approach is intentionally lightweight.

- Global design tokens live in `src/styles.css`.
- Page-specific layout and presentation live beside each page component.
- Shared login/signup form styles live in `src/app/pages/shared-auth.css`.
- The app uses restrained colors, stable spacing, and responsive layouts.

No UI component library is installed yet. This keeps styling flexible while the application shape is still being discovered.
