# Owner Vision And Context

This file is for high-level owner context that helps AI agents understand the bigger picture behind the product, technical decisions, learning goals, and working style.

Use this for thoughts that are broader than a single feature, story, prototype, or architecture decision.

Agents should read this for context, but should not create implementation stories directly from it. When a high-level idea becomes actionable, agents should first stage an interpretation under `docs/ai-agents/staging/` for owner review.

## How To Use This

Write freely. This file can include rough thoughts, philosophy, constraints, motivations, and long-term goals.

Useful entry format:

```text
## YYYY-MM-DD

What I am thinking:

Why this matters:

What I want agents to remember:

Open questions:
```
